#  Animated 3D Sports Car Book using R3F
![Screenshot](R3Fcar.png)

## How to use

1. Clone this repository

   ```bash
   git clone https://github.com/Hugosh71/R3F-SportsCar-Book.git
   ```
   
2. Install the dependencies

   ```python
   yarn
   ```

3. Launch 
   ```bash
   yarn dev
   ```
